## 一、如何运行该项目？

首先 clone 该项目

```
git clone git@github.com:Travelerv/cloudMusic.git
```

进入该项目后，安装依赖

```
npm install
```

运行该项目

```
npm run serve
```

> 注：如果数据加载不出来，可能是接口更换。
